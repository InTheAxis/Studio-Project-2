///////////////////////////////CAMERA///////////////////////////////
W, A, S, D : Move in the view direction or strafe
SPACEBAR : Move up along y-axis ONLY if no-clip is active
LSHIFT : (Hold to) Crouch, or move down along y-axis when no-clip is active
UP, RIGHT, DOWN, LEFT : Rotate camera about its axis (yaw, pitch) if mouse not captured
/////////////////////////////INTERACTIONS/////////////////////////////

/////////////////////////////////MISC////////////////////////////////
TAB : Show/Hide Axes, FPS and Mouse Cursor
RSHIFT (With Axes and Fps shown) : Change skybox
1, 2 : Cull back faces
3, 4 : Fill/Wireframe
5: Toggle No-Clip
6 : Toggle Light
7: Capture/Release Mouse and Hide/Show Cursor
R : Reset all
